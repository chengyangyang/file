一、 JSON 和字符串之间的相互转化？
	var str ='{"name":"huang","age":"23"}'   每个属性都必须用双引号
	1、JSON .parse(str);   这个是从一个字符串中解析出json对象；
	  1.1、上面得到的结果为   Object  age:"23"  name:"huang" __proto__:Object
	2、JSON .stringify(); 这个用于从一个对象解析出字符串。
	2.1、如：var a = {a:1,b:2} 结果：JSON .stringify(a) -----"{"a":1,"b":2}"
